vue-formidable
--------------

*status: unstable*


[Vue](http://vuejs.org) components to generate HTML forms from [JSON schemas](http://json-schema.org).


```js
var Form = require('vue-formidable')
Vue.component('vue-form', Form)
```

```html
<vue-form schema="{{someJSONSchema}}" value="{{x}}"></vue-form>
```
